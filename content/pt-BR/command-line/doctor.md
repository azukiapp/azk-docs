## azk doctor
Shows an analysis of the health of `azk`

#### Opções:

- `--logo`    Exibe o logo do azk antes de exibir a situação do sistema (padrão: falso)

#### Uso:

    $ azk [options] doctor [options]

#### Exemplos:

```
$ azk doctor

      Azk   : 0.8.3
      Agent : up
      Docker: 1.4.1
      Use vm: no
```
