## azk doctor

Shows an analysis of `azk`'s health.

#### Options:

- `--logo`    Shows azk logo before displaying the current system health status (default: false)

#### Usage:

    $ azk [options] doctor [options]

#### Example:

```
$ azk doctor

      Azk   : 0.8.3
      Agent : up
      Docker: 1.4.1
      Use vm: no
```
