## azk doctor

Mostra uma análise da saúde do `azk`. 

#### Opções:

- `--logo`    Exibe o logo do azk antes de exibir a situação do sistema (padrão: falso)

#### Uso:

    $ azk [options] doctor [options]

#### Exemplo:

```
$ azk doctor

      Azk   : 0.8.3
      Agent : up
      Docker: 1.4.1
      Use vm: no
```
